# CSS Grid + Flexbox Layouts Este proyecto muestra cómo combinar CSS Grid y Flexbox para crear layouts modernos y responsivos. Es un recurso complementario a los posts sobre [Flexbox](https://www.femcodersclub.com/recursos/css/flexbox) y [CSS Grid](https://www.femcodersclub.com/recursos/css/css-grid) de FemCoders Club. ## Estructura del Proyecto ``` css-grid-flexbox-layouts/ ├── index.html # Landing page principal ├── css/ │ ├── styles.css # Estilos generales │ ├── grid.css # Componentes usando Grid │ ├── flexbox.css # Componentes usando Flexbox │ └── combined.css # Componentes que usan ambos ├── js/ │ └── main.js # JavaScript básico para interactividad ├── images/ # Imágenes del proyecto └── examples/ # Ejemplos adicionales ├── dashboard/ # Ejemplo de dashboard ├── portfolio/ # Ejemplo de portafolio └── product-page/ # Ejemplo de página de producto ``` ## Tecnologías Utilizadas - HTML5 - CSS3 (Grid y Flexbox) - JavaScript vanilla - Diseño responsivo sin frameworks ## Características Este proyecto demuestra: - Cómo estructurar una landing page moderna con header, hero section, características, testimonios, pricing y footer - Uso de Grid para el layout principal - Uso de Flexbox para componentes internos - Navegación responsiva usando Flexbox - Sistema de tarjetas con Grid y contenido alineado con Flexbox - Secciones alternadas que usan diferentes combinaciones de Grid y Flexbox - Media queries para adaptación responsiva en diferentes dispositivos ## Guía de Implementación Cada sección del proyecto incluye comentarios detallados explicando qué técnica se está utilizando y por qué, haciendo que este proyecto sea también educativo. ## Cómo Contribuir 1.
