CSS Grid + Flexbox Layouts

Este proyecto muestra cómo combinar CSS Grid y Flexbox para crear layouts modernos y responsivos. Es un recurso complementario a los posts sobre Flexbox y CSS Grid de FemCoders Club.

Estructura del Proyecto

css-grid-flexbox-layouts/
├── index.html              # Landing page principal
├── css/
│   ├── styles.css          # Estilos generales
│   ├── grid.css            # Componentes usando Grid
│   ├── flexbox.css         # Componentes usando Flexbox
│   └── combined.css        # Componentes que usan ambos
├── js/
│   └── main.js             # JavaScript básico para interactividad
├── images/                 # Imágenes del proyecto

Ejemplos Prácticos

A continuación te mostramos algunos ejemplos de cómo se usa Grid y Flexbox en este proyecto:

Estructura General

Grid para Layout

/* CSS Grid para estructura principal */
.hero-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: var(--space-xl);
  align-items: center;
}

.features-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: var(--space-lg);
}

Flexbox para Componentes

/* Flexbox para componentes internos */
.feature-card {
  display: flex;
  flex-direction: column;
  align-items: center;
  text-align: center;
  padding: var(--space-lg);
}

.header-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Combinación Inteligente

Layout Holy Grail

/* Grid para layout principal */
.holy-grail-demo {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar content aside"
    "footer footer footer";
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

Navegación Responsiva

/* Flexbox para navegación */
.main-nav .nav-list {
  display: flex;
  gap: var(--space-lg);
}

/* Adaptación móvil */
@media (max-width: 768px) {
  .main-nav.active .nav-list {
    flex-direction: column;
    gap: var(--space-md);
  }
}
